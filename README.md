# ENDG-233
Collection of portfolio projects from ENDG 233 (Programming with Data), covering programming concepts such as control flow, functions, data structures, file handling, pseudocode, flowcharts, as well as hardware (Raspberry Pi Pico).

PORTFOLIO PROJECT #1:
Designed a simulation that models the completion time of a car in a speed race. The program:
  (1) Accepts user inputs about car characteristics and race conditions 
  (2) Computes acceleration and speed profiles 
  (3) Calculates total race time under the selected parameters. 
This project demonstrates structured program design, conditional logic, and numerical computation.

PORTFOLIO PROJECT #2:
Built a terminal-based application for analyzing residential property data.
Core features include:
  (1) Searching through provided property datasets
  (2) Performing calculations (averages, totals, mins, max, etc.)
  (3) Printing formatted, user-friendly summaries.
This program demonstrates skills in data structures, file I/O, and modular program design.

PORTFOLIO PROJECT #3:
Created three CSV files containing statistics related to the Palestinian genocide and developed a program to:
  (1) Read and process casualty, injury, and infrastructure loss data
  (2) Output structured summaries based on user queries
  (3) Handle real-world style datasets and reporting.
This project demonstrates data processing, file handling, and working with self-built datasets.

HARDWARE PROJECT PT 1- MORSE CODE:
A MicroPython program that continuously transmits a given message in Morse code using the Pico’s built-in LED. The code handles correct timing for dots, dashes, letter spacing, and word spacing. During demo, the board is connected to a hardware decoder used to verify accurate signal timing.

HARDWARE PROJECT PT 2- LED LIGHT TRANSMISSION:
A MicroPython program that sends a repeating RGB colour pattern to the hardware decoder using a structured format. Each pattern begins and ends with S, includes brightness values for R, G, and B (0–255), and a D value that sets how long the colour is displayed. The decoder reads the pattern and shows the corresponding light output.
